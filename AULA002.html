Exemplos de alinhamento e definição de classes

HTML e CSS são linguagens de conteúdo

  • HTML define os elementos e conteúdo geral da página
  • CSS define as formatações e estilos de cores, fontes etc.
  • Java Script inclui a programação de funcionalidades específicas

O HTML trabalha com elementos

  • Cada elemento possui tags
  • Entre as tags colocamos conteúdo
  • para alterar os elementos usamos atributos

O CSS formata elementos

  • O CSS tem seus prórpios atributos
  • Vários atributos podem ser agrupados para cada classe
  • Um elemento pode ter uma ou mais classes CSS
